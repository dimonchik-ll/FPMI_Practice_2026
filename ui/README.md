# Разработчик 4 — ui

Можно менять только `ui/` и `assets/ui/`.

UI не импортирует `core/`, `towers/` или `enemies/`. Он принимает `GameSnapshot` на отрисовку и возвращает `UiAction` из обработчика клика.

Экономика находится в `ui/economy.py`. Дальше можно добавлять меню, подсказки, паузу, окно победы и оформление.

Проверка: `python -m ui.sandbox`.
